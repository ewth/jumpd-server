1. Login as root

        sudo su

2. Install git:

        yum -y install git

        
3. Clone `jumpd-server` repo:

        git clone https://Huebacca@bitbucket.org/Huebacca/jumpd-server.git


4. Run script:

        bash jumpd-server/setup.sh

        Just use get-jumpd.sh
